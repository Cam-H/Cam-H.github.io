Robotics

During highschool, I was the programming lead for my team's FIRST robotics team wherein we programmed large robots in Java. To get an idea of the kind of stuff I worked on visit the FIRST website.

Currently, I am developing a program which utilizes accelerometer / gyroscope sensors to enable control of an electronic device using strictly head movements. The project uses a Raspberry Pi hooked up to an MPU6050 sensor to monitor the user's orientation in space. To see the source code for this project visit: https://github.com/Cam-H/Pi-Robotics/tree/master/src/sensors
