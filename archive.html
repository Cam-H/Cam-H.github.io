SYR - An exploration into game engine development

I got started working with C++ in working through the tutorial series developed by TheCherno.

I developed this project with the intent to understand how game engines work through the experience of an experienced game engine developer. While I eventually moved on to other projects, this represented by initial introduction to several important technologies and techniques:

  • Using OpenGL to interface with the GPU
  • Utilizing GLSL to produce shaders for diffuse and specular lighting
  • Application of stencil testing for cell shading and other features
  • Applying FreeType in the rendering of text
  • Direct manipulation of meshes, and in particular how to develop custom mesh file loaders
  • Use of entity component systems (ECS) like entt, a more efficient alternative to standard inheritance

Although I did not end up developing SYR for a specific purpose, the understanding developed was critical for later projects like AutoCarver.

Conway's Game of Life

Based on the cellular automata developed by John Horton Conway. A set of simple rules give rise to rather interesting effects in a grid-based map. In each generation cells are treated with the following rules:

  1. Any live cell with < 2 neighbors dies
  2. Any live cell with 2 or 3 cells survives
  3. Any live cell with > 3 neighbors dies
  4. Any dead cell with exactly 3 neighbors becomes live
